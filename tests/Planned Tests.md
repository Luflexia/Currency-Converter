# Тестирование приложения "CurrencyConverter"

## Запланированные входные воздействия:

### Аутентификация и регистрация
1. Ввод корректных данных при входе
2. Ввод некорректных данных при входе
3. Ввод корректных данных при регистрации
4. Ввод некорректных данных при регистрации
5. Попытка регистрации с уже существующим логином
6. Нажатие на кнопку выхода из аккаунта (вход в аккаунт выполнен)

### Конвертация валют
7. Ввод суммы для конвертации (целое число)
8. Ввод суммы для конвертации (дробное число)
9. Ввод некорректной суммы для конвертации (отрицательное число)
10. Ввод некорректной суммы для конвертации (нечисловые символы)
11. Выбор валюты для конвертации из списка
12. Попытка конвертации без выбора валюты

### Управление списком валют
13. Нажатие на кнопку добавления валюты
14. Выбор валюты из списка доступных валют
15. Удаление валюты свайпом влево
16. Перемещение валюты в списке (drag and drop)
17. Попытка добавления уже существующей в списке валюты
18. Добавление максимально допустимого количества валют

### Выбор банка и обновление курсов
19. Нажатие на кнопку выбора банка
20. Выбор банка из списка доступных банков
21. Попытка обновления курсов при отсутствии интернет-соединения

### Проверка устойчивости
22. Проверка работы приложения при потере интернет-соединения
23. Проверка сохранения данных при неожиданном закрытии приложения
24. Проверка корректности работы при быстром переключении между экранами
